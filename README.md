## Setup

1. Clone the repository:

   git clone https://github.com/mohamedtarek64/AI-Powered-Recipe-Generator-Complete-Project-Brie.git
   cd AI-Powered-Recipe-Generator-Complete-Project-Brie

2. Install dependencies for backend and frontend (adjust commands to your project):

   # Backend
   cd backend
   npm install

   # Frontend
   cd ../frontend
   npm install

3. Create a .env file (example):

   OPENAI_API_KEY=your_api_key_here
   DATABASE_URL=postgres://user:pass@localhost:5432/recipes

4. Run the project:

   # Backend
   npm run dev --prefix backend

   # Frontend
   npm start --prefix frontend

## Usage

- Open the frontend in your browser (http://localhost:3000 by default) and enter ingredients or preferences to generate recipes.
- Use the API endpoints documented in the backend to interact programmatically.
